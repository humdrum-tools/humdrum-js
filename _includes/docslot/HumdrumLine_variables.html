text — The pre-parsed text for the data line stored in the object.
fields — Data fields of line (tokens or global/reference records).
nextLine — Reference to the next HumdrumLine in the file.
prevLine — Reference to the previous line in the Humdrum data.
hasSpines — Boolean for if the line has token data.
lineType — The basic type of line.
owner — Reference to the HumdrumBase object that contains this object.
